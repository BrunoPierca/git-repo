Git Commands


    Basic Commands

  • $ git init Initiates a repository
  • $ git add fileName Adds a specific file to your repository
  • $ git status Checks status of the work tree
  • $ git commit Commits to local
  • $ git push 'yourRemoteName' Pushes to remote repo
  • $ git pull Pulls changes from remote to local repo
  • $ git clone Clones a specific repository

    Other commands

  • $ git config --global user.name 'yourUsername' Configurates your Username
  • $ git config --global user.email 'yourEmail' Configurates your Email
  • $ git rm --cached fileName Deletes your file from the repo
  • $ git add *.html Adds all your .html files to the repo
  • $ git add . Adds all your files to the repo
  • $ git commit-m 'Comment on the commit' Speeds up the commit as you can comment right in the Git console
  • $ git touch Creates a file, for example, .gitignore
  • $ git checkout 'nameOfTheBranch' Switch between branches
  • $ git merge 'branchName' Merge branches
  • $ git remote add 'name' 'yourURL' Sets a remote repo
